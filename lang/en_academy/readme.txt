Academy Moodle Language Pack

This directory contains the Academy customised Moodle language files
for English (UK).

To use this language pack:

 1) Copy them in your 'dataroot/lang/en_academy' directory.

 2) Open Moodle > Site administration > Language > Language settings and 
    change the Default language to "en_academy".

 3) Purge all caches.

For more information, see the Moodle Documentation:

   http://docs.moodle.org/en/Translation
